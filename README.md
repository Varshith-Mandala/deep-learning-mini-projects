# 🚀 Deep Learning Mini Projects

Welcome to my Deep Learning Mini Projects repository!
This repository contains a collection of deep learning experiments, model implementations, training pipelines, evaluations, and research-oriented mini projects across multiple domains.

The purpose of this repository is to practice, experiment, and strengthen practical understanding of Deep Learning concepts through hands-on implementation and model evaluation.

---

# 📌 Repository Overview

This repository includes:

* Neural Network implementations
* CNN-based image classification projects
* Transfer Learning experiments
* NLP and Transformer-based models
* Medical AI projects
* Model evaluation and comparison notebooks
* Training and inference pipelines
* Data preprocessing workflows

Each project focuses on understanding model behavior, training strategies, evaluation metrics, and practical deep learning workflows.

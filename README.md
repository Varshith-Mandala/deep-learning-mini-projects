# Deep Learning Mini Projects

## Overview

This repository contains small **deep learning mini projects** implemented using **PyTorch**.
The goal is to practice core deep learning concepts by building simple end-to-end pipelines.

Each project focuses on understanding:

* Data preprocessing
* Model architecture
* Training loops
* Evaluation

---

## Projects

### 1. MNIST Digit Classification

A neural network that classifies handwritten digits (0–9) using the MNIST dataset.

Main concepts covered:

* Data transforms
* Dataset and DataLoader
* Neural network implementation
* Training and evaluation pipeline

---

## Tech Stack

* Python
* PyTorch
* Torchvision

---

## Purpose

This repository is mainly for **learning and experimentation**, helping build a strong foundation in deep learning by implementing small practical projects.
